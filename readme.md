# Guild of Calamitous Intent

This is a collection of navigational tools and libraries.

- `gciSensors` [C++]
    - `LIS3MDL`: magnetometer
    - `LSM6DSOX`: accelerometer, gyroscope
    - `BMP390`: pressure
    - `libi2c`: Linux and Arduino
- `gciNavigation` [python, C++]
    - `Mahoney`
    - `Madgwick`
    - `ComplementaryFilter`
